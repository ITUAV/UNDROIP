Registraion Description

In the main folder
------------------

* main_readXTFFiles.m: main code to read xtf data

* main_detection.m: main code for detection

* main_registration.m: main code for registration


Normally, only these codes should be modified (for loading data or for parameters)


For each main code, there is a section named "parameters". It's only this section which should be modified.

Generally, for reading or saving data, the operator chooses the path and the name of the file.

If necessary, other parameters can be changed in this section



folder "functions"
------------------

Containing all needed functions for running the codes. 

This folder should add to the set path of Matlab.
